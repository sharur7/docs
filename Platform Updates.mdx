## March, 2025

<Update label="Mar 16" description="Feature">
  **API Cost Estimator**  
  
  API Cost Estimator helps users forecast and plan their model usage expenses before deploying or adding credits. With the cost estimator, users can:

  * Select any model to simulate expected usage costs
  * Input estimated daily input and output token usage
  * View real-time cost projections for daily and monthly usage
  * Understand base pricing for selected models (input/output token rates)
  * Get smart recommendations for credit top-ups based on usage patterns
  * Make informed decisions on model selection and scaling before deployment

</Update>

## November, 2025

<Update label="Nov 15" description="Feature">
  **Bring Your Own Model**  
 
  BYOM (Bring Your Own Model) enables users to seamlessly deploy and run their own models on the platform with full flexibility and control. With BYOM from Hugging Face, users can:

  * Select any model from Hugging Face by providing the model ID
  * Authenticate securely using a Hugging Face access token
  * Deploy models on GPUs of their choice based on performance needs
  * Run custom models alongside native platform models
  * Scale deployments based on workload requirements
  * Quickly move from model selection to production-ready deployment

</Update>

## October, 2025

<Update label="Oct 28" description="Launch">
  **Model Playground**  
 
  Playground is a unified interface that allows users to test and experiment with all models and APIs from a single page. With Playground, users can:

  * Evaluate model performance using custom prompts
  * View key metrics such as TTFT, TPS, and total tokens
  * Adjust model parameters to fine-tune outputs
  * Generate API keys and copy API requests directly
  * Configure system prompts and include few-shot examples
  * Enable or disable streaming directly from the UI

</Update>

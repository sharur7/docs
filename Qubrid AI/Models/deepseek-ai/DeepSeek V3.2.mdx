### Python

```python
from openai import OpenAI

# Initialize the OpenAI client with Qubrid base URL
client = OpenAI(
    base_url="https://platform.qubrid.com/v1",
    api_key="QUBRID_API_KEY",
)

# Create a streaming chat completion
stream = client.chat.completions.create(
    model="deepseek-ai/DeepSeek-V3.2",
    messages=[
      {
        "role": "user",
        "content": "Explain quantum computing in simple terms"
      }
    ],
    max_tokens=8192,
    temperature=1,
    top_p=0.95,
    stream=True
)

# If stream = False comment this out
for chunk in stream:
    if chunk.choices and chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
print("\n")

# If stream = True comment this out
print(stream.choices[0].message.content)
```

### Go

```go
package main

import (
	"bufio"
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://platform.qubrid.com/v1/chat/completions"

	data := map[string]interface{}{
		"model": "deepseek-ai/DeepSeek-V3.2",
		"messages": []map[string]string{
			{
				"role":    "user",
				"content": "Explain quantum computing in simple terms",
			},
		},
		"temperature": 1,
		"max_tokens":  8192,
		"stream":      true,
		"top_p":       0.95,
	}

	jsonData, _ := json.Marshal(data)
	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonData))
	req.Header.Set("Authorization", "Bearer QUBRID_API_KEY")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	res, _ := client.Do(req)
	defer res.Body.Close()

	scanner := bufio.NewScanner(res.Body)
	for scanner.Scan() {
		line := scanner.Text()
		if line != "" {
			fmt.Println(line)
		}
	}
}
```

## Live Example

> **Prompt:** `Explain quantum computing in simple terms`

**Response:**

```
Quantum computing is a fundamentally different way of processing information
— one that exploits the laws of quantum mechanics rather than classical physics.

Classical computers store data as bits: each bit is either 0 or 1.

Quantum computers use qubits, which can exist as 0, 1, or any combination
of both simultaneously — a property called superposition. This allows a
quantum computer to evaluate an enormous number of possible states at once,
rather than one at a time.

Two key properties make this possible:

  Superposition   — a qubit holds multiple values simultaneously until measured
  Entanglement    — two qubits can be correlated so that the state of one
                    instantly determines the other, enabling massively
                    parallel computation across linked qubit pairs

Where quantum computers outperform classical machines:

  • Cryptography        — breaking and constructing encryption systems
  • Drug discovery      — simulating molecular structures at atomic resolution
  • Optimization        — solving logistics, scheduling, and financial problems
  • AI acceleration     — certain training and inference workloads

The important caveat: quantum computers aren't general-purpose replacements
for classical computers. They are precision instruments — extraordinarily
powerful for specific problem classes that are intractable for even the
fastest supercomputers today.
```
